1. URL ID Parameter Manipulation

2. Price Manipulation
ini tuh sederhana banget, ceritanya ada website beli tiket harganya 10$ dan saldo user $1000 dolar. nah disini kita bisa manipulasi atau ubah ubah harga tiket melalui request user lewat burpsuite, di requestnya terlihat ada amount harga ticket nya 10, dan ini bisa diubah.

3. Changing Password
ini juga sederhana banget, dimana ini tuh seperti fitur change password, tpi saat user ganti password dan requestnya di tahan di burpsuite, maka akan kelihatan password dan user id nya, dari user id ini kita bisa mengganti password orang lain.
di videonya user name kita adalah sebagai christoper, tapi saat ubah user id jadi 2 di burpsuite, yang terganti malah passwordnya piere
